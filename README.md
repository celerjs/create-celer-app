# create-celer-app

Create an app powered by [celer](https://github.com/cyea/celer).

```bash
$ npm init celer-app <project-name>
$ cd <project-name>
$ npm install
$ npm run dev
```

If using Yarn:

```bash
$ yarn create celer-app <project-name>
$ cd <project-name>
$ yarn
$ yarn dev
```

## Templates

Available templates:

- `default` (default)


To scaffold with specific template:

```bash
$ npm init celer-app my-app --template xxxx
```
